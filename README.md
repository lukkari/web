# LUKKARI

Custom schedule of the ICT-building of TUAS

*(work in progress)*

## Install

1. `git clone https://github.com/zaynetro/lukkari.git`
2. `cd lukkari/app`
3. `mkdir logs`
4. `mkdir cache`
5. `cp config/config.example.js config/config.js`
6. `cd ../`

## Edit database info

Database connection data is in `app/config/config.js`

## Run

`gulp` and `node app`
