# LUKKARI

Custom schedule of the ICT-building of TUAS

*(work in progress)*

## Install

1. `git clone https://github.com/zaynetro/lukkari.git`
2. `cd lukkari/app`
3. `mkdir logs`
4. `mkdir cache`
5. `cp config/config.example.js config/config.js`
6. Edit *config.js* file with your parameters
7. `cd ../`
8. `gulp jade`
9. `gulp templates`
10. `gulp scripts`

## Run

`node app`
